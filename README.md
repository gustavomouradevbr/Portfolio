# Gustavo Moura — Portfolio

This repository contains the personal portfolio website for Gustavo Moura. The site showcases a hero, about, skills, projects carousel, and contact/footer sections. It was built with plain HTML, CSS and JavaScript, with UI designs and prototypes created in Figma.

## Features

- Responsive single-page portfolio
- Dark / Light theme toggle (preference saved in localStorage)
- Projects carousel with project cards and "Coming Soon" placeholders
- Modal popup for links/features that are not yet available
- Modular CSS (section files imported by `css/index.css`)

## Technologies

- HTML5
- CSS3 (modular files)
- JavaScript (vanilla)
- Figma (design and mockups)

## Repository Structure

- `index.html` — Main HTML file
- `script.js` — JavaScript for carousel, theme toggle and modal
- `css/` — Folder containing modular CSS files (`index.css`, `home.css`, `about.css`, `skills.css`, `projects.css`, `footer.css`, `light-mode.css`)
- `assets/` — Images and icons used by the site

## README (Português)

Este repositório contém o site portfolio pessoal de Gustavo Moura. O site apresenta uma seção principal (hero), About, Skills, carrossel de projetos e rodapé/contato. O projeto foi desenvolvido com HTML, CSS e JavaScript puro, e as telas e protótipos foram elaborados no Figma.
